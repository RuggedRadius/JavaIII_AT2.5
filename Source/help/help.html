Usage

Introduction

The list on the left side of the window contains the task master list. The list on the right is the current to do task list.


Add Tasks

Drag and Drop tasks from the left side list to the white box in the middle of the window labelled "Drop here to add to list". This will add the task to the current to do task list.


Clear All Tasks

To clear the current to do task list click "File" -> "Reset List". This will remove all items from the list and also reset the background art.
